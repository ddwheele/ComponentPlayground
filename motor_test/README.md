Motor Test:

Use with 
 * brushless DC motor (5V)
 * ESP32
 * Adafruit TB6612 1.2A DC/Stepper Motor Driver Breakout Board
 
Connect:
* ESP 3V3 - Power (3V)
* ESP GND - Ground
* ESP D4 - TB6612 AIN1
* ESP D5 - TB6612 AIN2
 
* TB6612 PWMA - Power (3V)
* TB6612 STBY - Power (3V)
* TB6612 GND - Ground (necessary?)
* TB6612 MOT A - Motor Positive

* Motor Negative - Ground
